# Python Log Analyzer

A simple Python-based log analyzer that reads log files and detects suspicious events.

## Features

* Reads log files
* Counts INFO messages
* Counts WARNING messages
* Counts ERROR messages
* Generates a log analysis report
* Displays suspicious events

## Technologies Used

* Python

## Sample Output

* Total Lines: 5
* Total Errors: 2
* Total Warnings: 1
* Total Info: 2
